PROJECT GOALS

The main goal of this project was to implement a digital solution for learning management of trainees. For me, it was to learn how to work on existing code and improve the functionalities with the help of design and programming and to learn the technology stack that was being used with project.

TECHNOLOGIES USED

C#
CSS
HTML5
AngularJS
Bootstrap
SQL Server
Model View Controller Framework
Windows Communicaton Foundation Framework


RESPONSIBILITIES

Working in Learner Team I focused on sketching and designing the requirement for front-end and focused on the implementation of functionalities to bring on front-end. After integration inside our team, we worked with other teams to merge the changes.

LEARNINGS

From this project I learnt about AngularJS, Bootstrap, Frameworks like WCF and MVC and many new technologies.I learnt how to work in a team, how to go through code, why coding guidelines are must to use and what makes integration the toughest part.Other than this I learnt how to work in fast paced enviornment where requirements can change at any time.
